# Marketing Funnel Intelligence

An interactive marketing analytics dashboard built using **Python**, **Streamlit**, **Pandas**, and **Plotly** to analyze customer conversion performance across different stages of a marketing funnel.

## Overview

Marketing Funnel Intelligence helps businesses understand how potential customers progress through the marketing funnel, identify conversion bottlenecks, evaluate campaign performance, and make data-driven decisions to improve overall marketing effectiveness.

This project was developed as part of **Future Interns – Data Science & Analytics Task 3**.

---

## Features

* Marketing Funnel Overview
* Conversion Rate Analysis
* Customer Drop-off Analysis
* Contact Channel Performance
* Monthly Campaign Analysis
* Job-wise Conversion Analysis
* Education-wise Conversion Analysis
* Executive Business Insights
* Growth Recommendations
* Interactive Dataset Preview

## Installation

Clone the repository

```bash
git clone https://github.com/Jyo-08/FUTURE_DS_03.git
```

Navigate into the project

```bash
cd FUTURE_DS_03
```

Install dependencies

```bash
python3 -m pip install -r requirements.txt
```

Run the dashboard

```bash
python3 -m streamlit run app.py
```
